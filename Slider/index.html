HTML5

Web sayfalarının yapısal ve içeriksel tanımlamalarını yapmaya yarayan bir işaretleme dilidir. Tarayıcılar tarafından yorumlanarak görüntülenir.

CSS

HTML ve diğer işaretleme dilleriyle web sayfalarının görünümünü ve düzenini belirten tarayıcı tarafından yorumlanan bir stildir.

BOOTSTRAP

Bootstrap, açık kaynaklı bir CSS çerçevesidir. Web geliştirme süreçlerini hızlandırmak için önceden tasarlanmış bileşenler, stil ve düzen sağlar.!

TAILWIND

Tailwind CSS, kullanıma hazır CSS sınıflarıyla hızlı ve özelleştirilebilir bir web arayüzü oluşturma kütüphanesidir. Bileşenleri hızlıca inşa etmeyi kolaylaştırır.

JAVASCRIPT

Tarayıcıda interaktif ve dinamik web sayfaları oluşturmayı sağlayan programlama dilidir. Hem istemci hem de sunucu taraflı uygulamalarda kullanılır.

REACT

Açık kaynaklı bir JavaScript kütüphanesidir. Kullanıcı arayüzlerini bileşen tabanlı, verimli kullanılabilir şekilde geliştirmek için kullanılır.

NODEJS

Açık kaynaklı bir çalışma zamanı ortamıdır. JavaScript'i sunucu tarafında çalıştırarak ölçeklenebilir ağ uygulamaları geliştirmeye olanak tanır.

PHP

PHP, sunucu taraflı, açık kaynaklı bir betik dildir. Web uygulamaları oluşturmak ve dinamik içerikleri işlemek için kullanılır. Popülerliği biraz azalmıştır.
